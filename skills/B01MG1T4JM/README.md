To use the Restaurant Explorer skill, try saying...

* *Alexa, Ask Restaurant Explorer to find nearest Chinese restaurants*

* *I want to know more details about Option one*

* *give me more options*

This application gives you the location information of nearest good restaurants in your location. You can start by asking the cuisine, then you can choose the restaurant options for location or else you can ask for more options.
